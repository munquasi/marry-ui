Add Customers

You can add a new customer in the system by following the below mentioned steps.

 

  1. In the home page of the application click the Contract Management tab.
    The Customers and Contract options will appear on the page.

    2.  

  2. Click on the Customers option. The Customers page will open with the list of added customers in the system.

  1. Click the Create New Customer button.

  1. Select the type of customer from the Customer Type screen. It can be a Person or an Organisation.

If Person is selected as Customer Type.

.

  1. Select the Person option in the Customer Type screen.
    The Create Customer screen will open with four different panels     Customer, Account, Account Address, and Address Purpose.

  2. In the Customer Panel, type the First Name and Last Name of the customer.

  1. In the Account Panel, type and select the Account Description and the Customer Class.

 

Note: By default the Account Type field will display EXTERNAL as account type. You can change it by selecting value from the drop-down list.

By default the Account Established field will display the current date. You can change it by selecting another date from the calendar attached

 

 

 

 

.

 

  1. In the Account Address panel, type the Site Name and select the Country from the available country list.
    On selecting Country the Address Line 1, 2 and 3, City, State, and Zip fields appears on the screen, where Address Line 1, City and Postal Code fields are mandatory.

  1. In the Address Purpose panel, click the Add Purpose button.
    A row is added to the panel to select the purpose of adding an address to an account.

  1. In the From field select the date from the calendar attached from which the purpose of adding the address will be effective.

  2. In the To field select the date from the calendar attached till which the purpose of adding the address will be effective.

  3. From the Purpose drop-down list select a purpose for adding the customer address in the system.

  4. In the From field select the date from the calendar attached from which the purpose of adding the address will be effective.

  5. In the To field select the date from the calendar attached till which the purpose of adding the address will be effective.

  6. From the Purpose drop-down list select a purpose for adding the customer address in the system.

  1. Click on the Save button to save the record.

 

  • Note: User can mark the purpose as Primary by selecting the Primary check box next to each purpose added.

    You can also remove the purpose by clicking the Remove Icon next to each purpose added.
