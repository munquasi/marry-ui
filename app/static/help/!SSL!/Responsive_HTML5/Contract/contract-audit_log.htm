Audit Log

An audit log is a security-relevant chronological record, set of records, and/or destination and source of records that provide documentary evidence of the sequence of activities that have affected at any time a specific operation. Audit log tab helps the user to track the transaction details of a contract for auditing purposes.

View audio logs

The details of the audit logs for a contract can be viewed from the contract module. It displays the unit charge, name and description of the selected audit log. It also displays the IP address and name of the person who modified the audit log.

 

To view Audit Logs:

 

  1. Click  icon on the Javelin Home screen and then select the Contract option .

 

 

The Recent Contract screen will appear displaying the list of all the recent contracts in the system.

 

 

  1. Click on any of the contracts among the list for which audit log needs to be viewed.   

The selected contract screen will open with the relevant contract details.


 

By default the system will show Details tab.

 

  1. Click the Audit log tab to display the list of all the audit logs related to the selected contract. The following screen will appear.

 

 

  1. Select the audit log for which you want to view the audit details.

The following pop up will appear on the screen displaying all the details about the slected audit logs.

 

 

The descriptions of the fields in View Audit Logs Screen are given below:

 

Field

Description

Change Description / Reason

Displays the reason for which the audit log was last modified.

Changed by

Displays the Email ID of the person who last modified the audit log.

IP address

Displays the IP address of the computer from which the audit log was last modified.

Date

Displays the date (with time) on which the audit log was last modified.

Action Type

A suitable name for the audit log to explain the action of the audit log.

Type

Field for which the given audit log is related to.

Action

Combines the information in the Action Type and the Type. The space is replaced by an underscore and they both are concatenated with an underscore too.

 

Example:

If Action Type: Edit Inventory Name

And Type: Inventory

Then Action:Edit_Inventory_Name_Inventory

Field Name

Displays the name of the fields for which the names before and after the modification will be shown. It has 3 fields:

●      Name

●      Decription

●      Unit charge

Before Change

Displays the information of the fields (mentioned in the Field Name column) before the audit log was modified.

After Change

Displays the information of the fields (mentioned in the Field Name column) after the audit log was modified.

 

  1. Click on Cancel button once the Audit log detail is viewed.
