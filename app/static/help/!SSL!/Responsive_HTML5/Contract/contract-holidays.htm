Holidays

The holidays tab helps the user to check the holidays in a particular contract. This data can be used for billing and tracking purposes. This tab has the feature to Edit a holiday defined for the selected contract. Basically it has 3 features to manage the holidays for an employee:

  • Billed: When the customer is billed for that selected holiday

  • Paid:  When the employee is paid for that selected holiday

  • Worked: The Billed and Paid tab will be automatically marked as checked once the user marks worked as checked.

Edit holidays at contract level

The holidays can be edited according to the requirement. Based on this data the customer is charged and the employee is paid.

 

To edit a holiday:

 

  1. Click  icon on the Javelin Home screen and then select the Contract option .

 

  

 

The Recent Contract screen will appear displaying the list of all the recent contracts in the system.

 

 

  1. Click on any of the contract among the list for which the rate card needs to be added.  

The selected contract screen will open with the relevant contract details.

  1. Click on Holiday tab. A list of all the holidays will be shown.

 

 

  1. Click on Edit button under the Action column to edit the selected Holiday.

 

 

  1. Edit the details of the Holiday .

 

User can select the AM and PM from the drop down menu of the time period tab.

 

  1. Put the check mark on the relevant field for Billed, Paid and Worked.

 

The user can individually select Billed or Paid or both. The Billed and Paid tab will be automatically marked as checked once the user marks worked as checked.

 

  1. Click on Save to save the changes and Cancel to cancel the changes.
