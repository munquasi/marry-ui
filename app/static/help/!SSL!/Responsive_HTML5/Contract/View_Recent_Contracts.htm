View Recent Contracts

 

The contract Manager or authorised user can view a list of recent contracts along with their various statuses.

 

To view recent contracts:

 

1.     Click on the  icon and then Contract menu as shown.

 

On clicking the Contracts menu, a list of recent contracts is displayed as shown below:

 

 

The screen shows a list of recent contracts along with their key details and various current status.

2.     Click on the desired contract record to view proper details of the contract.

On clicking any record, the complete details of the contract is displayed:

 

3.     Click on the  button to edit the details of the contract.

4.     Once the changes are made (if required), click the button to proceed with the changes.

 

 

The use may require authentication and edit rights over the information to make the required changes. For the required rights, the user can contact the concerned administrator.
