Catalogue

Catalogues allows the user to create and manage various components of a contract such as:

●      Rate Card

●      Equipments

●      Allowances

●      Direct Management 

 

These tabs are explained below separately. The details in these components are helpful for billing purposes. Based on the information in these tabs, the customer is charged and the employees are paid.

 

 

To view Catalogue tab:

 

  1. Click  icon on the Javelin Home screen and then select the Contract option .

 

The Recent Contract screen will appear displaying the list of all the recent contracts in the system.

 

 

  1. Click on any of the draft contract among the list for which the rate card needs to be added.  

 

User should select a contract with status as Draft to create a Rate Card.

 

The selected contract screen will open with the relevant contract details. By default the system will show Details tab.

 

  1. Click the Catalogue tab. The Catalogue screen will appear with the following tabs:

●      Rate Card

●      Equipment

●      Allowance

●      Direct Management.
