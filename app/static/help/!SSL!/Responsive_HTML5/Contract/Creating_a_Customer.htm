Creating a Customer

 

Users can create a new customer record in the OPS system. This allows the user to create contracts and contract associated details for the customer in the finance system. Customer records are mastered and managed in Oracle ERP. It has two different types:

●      Organisation:  A customer can be created as an organization

●      Person: A customer can be created as Person. The created person can be later used as a contact.

 

Each Customer Record can have one or more Account and each Account can have one or more Account Profile. For the created customer, the user can fill in the following related details:

●      Sites: A customer account can have multiple sites

●      Contracts: A customer can have multiple contracts.

●      Contacts: A customer account can have multiple contacts

 

Finance users would then complete the full Customer details like adding of Account, Account Profile and Sites and contract and so on.

Ops-2-2.1.3 Create Customer Account

By default, a customer account is automatically created when the user creates a new customer. Once the user creates a customer account, then for the account:

●      Site can be created

●      Contact and Contact point details can be created

●      Contract linked to the account can be created

Ops-2-2.1.3.1 Add Account for a Customer

 

When the user creates a customer, a default account associated with the customer is automatically created for which the user can link it to a site, contact and contract.

 

If the user requires any additional account for handling a number of sites, and contracts etc, then the user can create it if required.
