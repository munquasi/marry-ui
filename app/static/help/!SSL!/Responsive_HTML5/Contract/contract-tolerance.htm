Tolerance

Tolerance tab is used to set specific tolerances for check in and check outs so that the user can manage tolerance at contract level and agree these with the client for the payment.

Add Tolerance

There is always a need to add the tolerance in the system to manage the billing.

 

To add a Tolerance:

 

  1. Click on the Tolerance Tab to display the list of all the tolerances for the given contract.

 

To view how to navigate to this screen, click here (click again to collapse).

Tolerance can only be setup in a draft contract. For additions to an active contract user must enter change control.

 

  1. Click on the Add Tolerance button to add a new Tolerance in the selected contract of the customer.  

Use the arrows to increase / decrease the timing.

By default the time shift time is in minutes. The following window will appear on the screen displaying all the required detailed to be filled.

 

 

  1. Fill the required details.

 

The descriptions of the fields on the Add Tolerance screen are given below:

 

Field

Description

Before shift start

Type the allowed tolerance before the shift starts.

After shift start

Type the allowed tolerance after the shift starts.

Before shift Ends

Type the allowed tolerance before the shift ends.

After shift Ends

Type the allowed  tolerance after the shift ends.

Before Call

Type the allowed  tolerance before the call starts.

After Call

Type the allowed  tolerance after the call starts.

Default

Makes the selected tolerance as default for the selected contract

 

The User can set the selected tolerance as Default by putting a check mark on the default tab.

 

 

  1. Click on Save to save the changes and Cancel to cancel the changes.
