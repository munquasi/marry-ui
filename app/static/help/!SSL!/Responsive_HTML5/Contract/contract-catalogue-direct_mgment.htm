Billing for Direct Management

At contract level it is possible to set up charges for Direct Management. Direct management feature under contract level is a sub section located within the catalogue. While setting up a direct management record, the user is prompted to select what the billing plan it is associated to. After Direct Management charge rates are set at contract level, they are available to be used for any location record created for a work order that has a matching billing plan. OnceDirect Management is assigned to a given location and the location has been activated, a script runs that picks all direct management (indirect labour) linked at customer locations and creates Billing Lines for the same in Ops. These billing lines are sent to Oracle PPM to create equivalent billing events.

 

Direct Management can only be set-up for Draft Contracts or via the change control approach when the contract is active. For new contract, a user must have created at least one Bill Plan to be able to create a direct management record.

 

The billing is checked under the Location tab after selecting a work order.

Assigning a Direct Management

The employees are paid and the customer is charged based on lot of parameters. Out of those direct management plays a very important role. For that, it is important to assign a direct management for whom the billing is to be charged from the customer.

 

To assign a Direct Management:

 

 

  1. Select the Work Order under which you want to see the direct management details for billing purposes.

The following screen will appear displaying the details of the selected work order.

 

  

 

  1. Click on the Location tab which will display the list of locations.

        The Location ID, Site name, Start date and End Date can be seen from the available list.

 

  

 

  1. Select the required location. The following contents at the location level will be displayed on the screen.

 

        

 

  1. Click on the Direct Management tab to display the list of the direct management (indirect labours) available for assigning to the selected location.

 

         

 

  1. Put a check mark next to the name of the Direct Management you need to assign for the selected location.

The selected  Direct Management will be assigned for the given Location.

 

      

The Direct Management (Indirect Labour) cannot be assigned when the Location is set to be Closed.

 

          

 

  1.  Click on the View tab to display the Information of the direct management (Indirect Labour) for billing purposes.

     The following screen will appear displaying the various information of the direct management.

The Bill Plan, Unit Charge, Customer Reference, Service Detail, start date and End Date can be seen after  clicking on View.

 

          

 

 

The descriptions of the fields on the Direct Management screen are given below:

 

Field

Description

Bill plan

Select  the bill plan for which the employee is working. This is defined in the Bill Plan tab under selected Contract.

Unit Charge

Type the unit charge as per the already entered information during the time of creating of the direct management. This unit charge will be paid only for one time depending upon the cycle assigned in the bill plan.

Customer Reference

Type the reference number useful for getting track of the customer details.

Service Detail

From the drop down menu, select the type of the guard explaining his work area. This is added during the time of creation of the direct management under the contract level.

Name

Type the name of the Direct Management. It becomes easier to differentiate between various direct management when assigned a name for each one of them.

Description

Briefly explain about the selected direct management.

Start/End Date

This data is fetched from the database for the selected contract.

 

 

      

The user can only view the direct management details from the location tab. To edit the Direct Management the user has to go to the Direct Management tab under the Catalogue tab after  selecting a contract.
