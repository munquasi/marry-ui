Ops-2.2-2.2.13 Tolerance

Tolerance tab is used to set specific tolerances for check in and check outs so that the tolerance can be managed at contract level and agreed with the client for the payment.

Ops-2-2.2.13.1 Add Tolerance

There is always a need to add the tolerance in the system to manage the billing. The employees are paid and the client is charged based on this information.

 

To add a Tolerance:

 

1.     Follow the steps 1-5 of the section Ops-2-2.2.1 Searching for a Contract. To reach the contract for which the tolerance need to be added.

 

Tolerance can only be setup in a draft contract. For additions to an active contract user must enter change control.

 

The selected Contract screen will open with the relevant contract details.

 

 

2.             Click on the Tolerance Tab to display the list of all the tolerances for the given contract.

 

 

3.             Click on the Add Tolerance button to add a new Tolerance in the selected contract of the customer.  Use the arrows to increase / decrease the timing.  By default the time shift time is in minutes. The following window will appear on the screen displaying all the required detailed to be filled.

 

 

4.             Fill the required details. The descriptions of the fields on the Add Tolerance screen are given below:

 

Field

Description

Before shift start

Type the allowed tolerance before the shift starts.

After shift start

Type the allowed tolerance after the shift starts.

Before shift Ends

Type the allowed tolerance before the shift ends.

After shift Ends

Type the allowed  tolerance after the shift ends.

Before Call

Type the allowed  tolerance before the call starts.

After Call

Type the allowed  tolerance after the call starts.

Default

Makes the selected tolerance as default for the selected contract

 

The User can set the selected tolerance as Default by putting a check mark on the default tab.

 

5.             Click on Save to save the changes and Cancel to cancel the changes.

 

The added tolerance will be displayed in the Tolerance list of the selected contract as shown below.

 

 

To view the details of the tolerance without opening it, the user can see the first column named as ‘NAME’ which displays everything under the selected tolerance.

 

Example:

When the following information is written under the ‘NAME’ column of the selected tolerance

 

 

then its description is as follows:

 

20 = Before shift start

25 = After shift start

20 = Before shift Ends

20 = After shift Ends

CC = Check Call Tolerance

10 = Before Call

10 = After Call

Once a tolerance is added, it can be edited according the the requirement.

 

To edit a Tolerance:

The selected Contract screen will open with the relevant contract details.
