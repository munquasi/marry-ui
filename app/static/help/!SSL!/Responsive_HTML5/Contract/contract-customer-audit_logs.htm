Customer Audit Logs

A single customer can have multiple audit logs, these audit logs are helpful for correct billing and payment purposes. The user can view audit logs for a selected customer via the Audit Logs tab under customer view screen.

 

To view audit logs:

 

  1. Click on the Audit Logs tab in the customer view screen.

The Audit logs will be displayed on screen as shown:

 

 

  1. Click on the audit log you want to view. The following pop up screen will appear.

 

 

The descriptions of the fields in View Audit Logs Screen are given below:

 

Field

Description

Change Description / Reason

Displays the reason for which the audit log was last modified.

Changed by

Displays the Email ID of the person who last modified the audit log.

IP address

Displays the IP address of the computer from which the audit log was last modified.

Date

Displays the date (with time) on which the audit log was last modified.

Action Type

A suitable name for the audit log to explain the action of the audit log.

Type

Field for which the given audit log is related to. Here it is CONTRACT.

Action

Displays the action taken for the audit log.

 

Example:

Edit_Contract

Save_Contract

Save_Contract_Site

Field Name

Displays the name of the fields for which the names before and after the modification will be shown. It has 3 fields:

●      Name

●      Decription

●      Unit charge

Before Change

Displays the information of the fields (mentioned in the Field Name column) before the audit log was modified.

After Change

Displays the information of the fields (mentioned in the Field Name column) after the audit log was modified.
