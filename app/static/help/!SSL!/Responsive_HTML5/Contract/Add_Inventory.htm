Add Inventory

To get the benefits of the Inventory, it is important to add a new inventory with the required fields. The inventory can be added in a contract under the Catalogue tab of the contract details.

 

To add an Inventory:

 

To view how to navigate to this screen, click here (click again to collapse).

 

  1. Click the  icon on the Javelin Home screen and then click Customers option.

 

  1. The following screen displays with the option to search for an existing customer. Search for any existing customer by typing the relevant keywords and clicking the  button.

 

  1. On the list of customers displayed, click the desired customer record (highlighted in red). The existing account(s) related to the customer is displayed on the right side of the screen as (highlighted in blue) below:

 

 

  1. Click on the account link highlighted in red (shown in the preceding figure). The details existing under the customer account just clicked is displayed in tabbed view.

 

  1. Click on Contract tab to display the contract screen with the list of existing contracts (if any) as shown.

 

 

  1. Click on any of the existing contracts to view the details of the contract.

 

 

  1. Click the Catalogue tab. The Catalogue screen will appear with the following tabs:

●      Rate Card

●      Equipment

●      Allowance

●      Direct Management.

By default the system will show Rate Card tab.

 

 

  1. Click on the Equipment tab under Catalogue.

 

 The Add Inventory and Add Assets button would be displayed with the list of equipments added (if any).

 

 

  1. Click the Add Inventory button in the current tab to add a new inventory.

The following screen will appear displaying all the information required for adding an inventory.

 

 

The descriptions of the fields on the Add Inventory screen are given below:

 

Field

Description

Name

Name of the equipment to be added as an inventory.

Description

A significant description of the equipment kept as an inventory.

Start Date

Captures the Start date for the Inventory.

End Date

Captures the End date for the Inventory.

Inventory ID

A suitable ID for the equipment to be kept as an inventory.

Unit Charge

Charge per unit of the equipment added as an inventory.

Max Quantity Supplied

The maximum quantity of the inventory supplied to the customer.

Service Code

A code for the equipment to which it needs to be linked.

Service Description

A significant description of the service code associated with the inventory. This field is not mandatory.

Customer Reference

A reference for the customer associated with the inventory.

This field is not mandatory.

Revenue Event

A record about the revenue details associated with the inventory.

Invoice Event

A record about the invoice details associated with the inventory.

 

  1. Click on Save once all the required details are entered.

The following pop-up will appear.

 

 

  1. When you click on OK, the system will show the relevant contract screen displaying the Details tab.
