Charge Rate:

Charge rate is the rate charged from the customer for the services provided by G4S. Based on this information the billing is done. It consists of 3 tabs as follows:

  • Standard Rate

  • Multiplier Rate

  • Fixed Rate

 

To add a Standard Charge Rate:

  1. Scroll down to reach Charge Rate tab and click the button to edit the Standard Charge Rate as shown below.

 

 

The following screen will appear. By default Standard is written under the Rate Name tab as shown below.

 

 

  1. Fill the Standard Rate (in Pounds) that you want to enter for the defined Rate Card. By default it is set to 0 Pounds.

  2. Enter the start time and end time for the standard charge rate. By default it is set to 00:00.

  3. Select the days of the week for which the standard charge rate need to be valid.

  4. Click update once all the details are filled. Click Cancel to cancel the update.

 

To add a Multiplier Charge Rate:

 

  1. Click on Multiplier Rate (if required) to get the multiplied charge rate for the given rate card. The following window will appear.

 

 

  1. Type any figure on the Multiplier Factor. The system will automatically reflect it under the Rate Name field and display the figure in the Rate field after multiplying by the standard charge rate. You can also manually change the Rate name.

 

Example: In the above screen-shot you can see that when 2 is typed in the multiplication factor then automatically the value in Rate field turns to be 24 pounds because the standard charge rate was 12 pounds.

 

  1. Click Add Line once all the details are filled. Click Cancel to cancel the update.

You can add more than one multiplier rate.

 

To add a Fixed Charge Rate:

 

  1. Click on Fixed Rate to add fixed rate for the given rate card. The following window will appear.

 

 

  1. Enter a suitable name for the fixed rate in the Rate Name tab.

  2. Fill the fixed rate in the Rate tab (in Pounds) that you want to enter for the defined Rate Card. By default it is set to 0 Pounds.

  3. Enter the start time and end time for the standard charge rate. By default it is set to 00:00.

  4. Select the days of the week for which the standard charge rate need to be valid.

  5. Click Add Line once all the details are filled. Click Cancel to cancel the update.

Break Management

Break management tab shows whether the customer is charged for the break given to the G4S employees or not. It also shows whether the G4S employees are paid for those breaks or not. There is also a feature of  limiting the time period of the break using the tabs Break Length and Break Entitled Over.

 

Breaks can be of following four types :

  • Billed to the Client but not paid to employee,

  • Billed to client and paid to employee

  • Not Billed to Client and not paid to Employee

  • Not Billed to Client and paid to employee

  • By default the break is Billed to client and paid to employee

 

 

To add a break management:

 

  1. In the Break Management tab select any one or both among the Paid or Billed check box. By default both check boxes is selected.

 

 

When Break is Paid and Billed, Break Length and Break Entitled Over fields are not required.

 

  1. Uncheck any one or both according to the requirement.

 

Uncheck Paid if the break is paid to employee but Not Billed to the Client.

 

 

Uncheck Billed if the break is Billed to the Client but not paid to employee.

 

 

Uncheck both if the break is Neither Billed to the client nor paid to the Employee

 

 

When break is Not payable and/or not billable then user is able to enter Break Length (in minutes). Default value for Break Length will be the value which will be set in country module.

 

  1. Enter the Break Length in minutes (if required to change).

 

The Break Length field will be added with Break Entitled Over (in hours) field which is fetched from the Country Module. (it will be blank if nothing is mentioned in the country module regarding break entitled Over). Break Length can't be less than that defined at country level.

 

  1. Enter the Break Entitled Over in hours (if required to change).

 

  1. Click Save to save the record or you can click the Save & Close button to save and navigate back to the previous screen.
