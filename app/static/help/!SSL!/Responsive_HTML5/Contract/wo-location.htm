View/Edit Customer Locations

In the Work Order tab, user can view and set the location of the work order using the site/location detail already existing in the Sites tab under Contract.

 

To view the location of a work order:

 

  1. Click on the Location tab in the Work Order screen.

To view how to navigate to the work order screen, click here.

  1. Click the  icon on the Javelin Home screen and then click Customers option.

 

  1. The following screen displays with the option to search for an existing customer. Search for any existing customer by typing the relevant keywords and clicking the  button.

 

  1. On the list of customers displayed, click the desired customer record (highlighted in red). The existing account(s) related to the customer is displayed on the right side of the screen as (highlighted in blue) below:

 

 

  1. Click on the account link highlighted in red (shown in the preceding figure). The details existing under the customer account just clicked is displayed in tabbed view.

 

  1. Click on Contract tab to display the contract screen with the list of existing contracts (if any) as shown.

 

 

  1. Click on any of the existing contracts to view the details of the contract.

 

 

 

  1. Click on the Work Order tab to view the existing list of Work orders under the contract.

 

 

  1. Click on any of the work orders listed on the screen to view the details of the work order.

 

 

 

 

  1. Click on any location record from among the list of locations displayed. The Location Summary screen is displayed.

 

 

The user can view edit or archive the information present on the Location Summary screen.

  • If the Customer Location record is no longer in use, the user has the option to archive the record by clicking the  button. Once archived, the particular Customer Location will become Read-only to the users.

  • Once archived, the user has the option to undo the archive action by clicking the button. Once Unarchive button is clicked, the Customer Location record will be active again.

To add a contract location for a work order:

  1. Click on the Location tab in the Work Order screen.

  1. Click the  icon on the Javelin Home screen and then click Customers option.

 

  1. The following screen displays with the option to search for an existing customer. Search for any existing customer by typing the relevant keywords and clicking the  button.

 

  1. On the list of customers displayed, click the desired customer record (highlighted in red). The existing account(s) related to the customer is displayed on the right side of the screen as (highlighted in blue) below:

  1. Click on any of the existing contracts to view the details of the contract.

The descriptions of the fields on theLocation Summary screen are given below:

 

Field

Description

Set Location

This is the location where the work order will be applicable. Click on the Set Location option, the pop-up to search the appropriate location will appear on the screen as shown below:

 

Once the user is able to search the required location, click on the button to set the location to the Location Summary screen.

Address

Mention the address of the location where the work order will be applicable.

Coordinates

Enter the text for coordinates of the location for the work order.

Start Date

Date from which the duties in the location will start. Displays the current date by default on the Start date field.

End Date

Date when the duties in the location will End. Displays the current date by default on the End date field.

To change/select another end date for the location, click on the icon, the calendar to select date will  get displayed as shown below:

 

Proof Of Duty

Allows the user to set the proof of attendance of duty by an employee such as site login, IVR, Reports etc. Displays IVR by default. To change the proof of duty, select the appropriate option from the drop down menu as shown below:

 

Once the user select any of the option from the drop down, the information will get displayed on the Proof Of Duty field.

Method Of Recording

Select the Method of Recording of duty attendance, by selecting the appropriate option from the drop down menu as shown below:

 

Once the user select any of the option from the drop down, the information will get displayed on the Method of Recording field.

Open Ended

When the option is selected, the end date of the duties will not be defined. Employees can continue with their duties without an end date. On clicking Open Ended  box, the End Date field will get hidden and the location is set as open ended.

Call In

It is the frequency of calls that occurs between the employee and the contract manager for status updates about the duty. On clicking Call In box, the Call In Frequency field will get displayed on the Location summary screen. The user can set the call frequency by selecting the option from the drop down menu as shown below:

 

Once the user select any of the option from the drop down, the information will get displayed on the Call-In-Frequency field.
