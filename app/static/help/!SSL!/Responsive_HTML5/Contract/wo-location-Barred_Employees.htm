Barred Employees

View/Manage Barred Employees for a Location

When a user has barred an employee from a site through a customer location record or from Site Record, then the employee is also barred from any new customer location record created for the same customer for the same site.

 

Therefore if new work order is raised to deliver services to the same site, these would automatically be reflected in any Customer location record assigned to that site and to the Site barred employee also. So Barred Employees are visible and manageable at both the site record as well as the Customer location.
