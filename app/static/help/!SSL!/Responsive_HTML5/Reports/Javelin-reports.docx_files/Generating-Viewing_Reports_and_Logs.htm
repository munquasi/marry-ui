Report Options

After the user generates a report, there are various icons and options available in the report interface which are common across the reports and can be used to do the following:

●    Print: Print the report in PDF or HTML format

●   Export: Export with option to select certain fields

●    Export report: Export the report in various formats

●   Print server: Print the report on the server

●    Report Parameters: Click this icon to run the report

●    Report TOC: Click to bring toggle table of contents for navigation.

Options for navigating the report:

●    First Page: Click to jump to the generated report’s first page

●    Previous Page: Click to navigate to the previous page relative to the current page

●   Next page: Click to navigate to the Next page relative to the current page

●   Last Page: Click to jump to the generated report’s last page

●    Navigate directly to specified page
