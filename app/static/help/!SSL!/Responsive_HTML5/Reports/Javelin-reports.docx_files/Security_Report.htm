Security Report

This report provides a log of system access done or attempted by various users of a department. It lists the details such as Date time stamp, IP address, Geolocation of the IP address etc to help the system trace any breach of the security by the users.

 

To access the Security violation Report:

  1. Go to Report Management and click on Audit Reports.

  1. Next, select Security Report from the Reports drop down list  and click Go.

  1. On the report filters that gets displayed, select or enter the desired filter criteria.

Some of the notable columns displayed in this report are:

Report Column

Description

Login Date

Date on which the access was attempted

Login Time

Time  on which the access was attempted

Username

User Name of the person attempting to access the system

Source IP

Source IP address of  the person attempting to access the system

Menu

Menu attempted to access

Browser/version

Browser and version used to attempt the access

Geolocation

Geographical location of IP address attempting the access

 

The file can be downloaded in csv format.

 

This report displays information for a span of one year.
