Scheduling Accuracy Report

This report shows shifts slots where employees are not yet assigned to the shifts. (It is an exception report that lists all the posts filled by actual scheduling when the regular scheduler failed.)

 

To view the Scheduling Accuracy Report:

  1. Go to Report Management and click on Operation Reports.

 

 

  1. Next, select Scheduling Accuracy Report from the Reports drop down list  and click Go.

 

 

  1. On the report filters that gets displayed, select or enter the desired filter criteria.

 

 

All fields marked with * are mandatory. Please ensure that the user provides information in these fields to successfully generate the report.

 

  1. After selecting the filter criteria, click Show Report. The report will get displayed similar to the one shown below:

Image will be updated

 

  1. You can scroll the report to the right and left if required.

 

Some of the notable columns in this report are:

Description

Unique ID of Work Order

Total permanent hours scheduled in advance for the period which were delivered by the same SO as scheduled (XX)

%

Percentage of hours delivered by employee planned to work shift

 

 

What are Advanced Scheduled Hours?

It is the number of hours worked by the same employee as scheduled 7 days prior to the commencement of the each shift.
