Employee Working Time Report

This report shows the user, the logged working time of employees which also provides information on Non-Compliance with Working Time Regulations, Number of staff exceeding working hours, Number of direct labour staff paid in month. The user can view a summary and detailed view of the report.

 

To access the Employee Working Time Report Report:

  1. Go to Report Management and click on Operation Reports.

 

 

  1. Next, select Employee Working Time Report from the Reports drop down list and click Go.

 

  1. On the report filters that gets displayed, select or enter the desired filter criteria.

 

All fields marked with * are mandatory. Please do not leave these fields blank to successfully generate the report.

 

The Start date and End date should be within a monthâ€™s range. And the End date should not be a future date.

Summary Report Columns:

Report Column

Description

Country

Country to which Employees belong

Department

Department to which the Employee belong

Employees Exceeding Hours

Number of employees that exceed working hours in period

Total Employees

Number of direct labour employees paid in period

Compliance %

Number of employees exceeding working hours as a percentage against total number of employees paid in period

 

Detailed Report Columns:

Report Column

Description

Country

Country to which Employees belong

Department

Department to which the Employee belong

Employee Number

Unique ID of Employee

Employee Surname/forename

Surname/forename of employee

Hours Worked

Hours employee has worked in period

Regulation Hours

Hours to which regulations allow employee to work(Working Time Option)

Variance

Variance of hours employee has worked in period to regulation hours
