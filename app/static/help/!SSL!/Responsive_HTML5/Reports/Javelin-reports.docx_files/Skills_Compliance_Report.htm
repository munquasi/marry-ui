Skills Compliance Report

This report displays compliance information of employees’ skills for the post on which the employee is being deployed.  It also displays information on Under-skilled hours delivered (The number of hours delivered in the period in which the SO deployed did not meet the skills required for the post).

 

To access the Skills Compliance report:

  1. Go to Report Management and click on Operation Reports.

 

 

  1. Next, select Skills Compliance Report from the Reports drop down list  and click Go.

 

 

  1. On the report filters that gets displayed, select or enter the desired filter criteria.

 

 

The Start date and End date should be within a month’s range. And the End date should not be a future date.

 

  1. After selecting the filter criteria, click Show Report. The report will get displayed similar to the one shown below:
