License Compliance Report

This report displays compliance information of employees’ licenses for the post on which the employee is being deployed.

To access the License Compliance report:

  1. Go to Report Management and click on Contract Reports.

 

 

  1. Next, select License Compliance Report from the Reports drop down list  and click Go.

 

 

  1. On the report filters that gets displayed, select or enter the desired filter criteria.

 

The Start date and End date should be within a month’s range. And the End date should not be a future date.

 

  1. After selecting the filter criteria, click Show Report. The report will get displayed similar to the one shown below:

Summary Report

Above image will be changed

 

Detailed Report

  1. You can scroll the report to the right and left if required.

 

Summary Report:

Report Column

Description

Un Licensed Hours

The number of hours delivered in the period in which the SO deployed did not meet the License required for the post on which he/she was deployed(BB)

Contracted Hours

Total of permanent and temporary hours in period(A)

Compliance%

Number of unlicensed hours shown as a percentage against total contracted hours

Un-Licensed Hours

The number of hours delivered in the period in which the SO deployed did not meet the License required for the post on which he/she was deployed (BB)

Department

Department to which the location belongs
