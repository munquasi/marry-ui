Audit Report

This report allows an a user to view the details of changes that has been done on an item including date time stamp, IP address, description and type of change etc.

 

To access the Audit Report:

  1. Go to Report Management and click on Contract Reports.

 

  1. Next, select Audit Report from the Reports drop down list  and click Go.

 

 

  1. On the report filters that gets displayed, select or enter the desired filter criteria.

 

 

The Start date and End date should be within a monthâ€™s range. And the End date should not be a future date.

 

 

  1. After selecting the filter criteria, click Show Report. The report will get displayed similar to the one shown below:

 

  1. You can scroll the report to the right and left if required.

 

Some of the notable columns displayed in this report are:

Report Column

Description

Change Type

Description of type of Audit/change

Country/department/Branch

Country/department/branch to which work order belongs

User ID

Unique ID of user who made change

User Name

Name of the user who made change

IP Address

External IP Address of device user used to make change

Date/TIme

Date/Time at which change was made

Type

Item to which change was applied

ID

Unique ID of item changed

Description

Name of item changed

Old Value

Value before change

New Value

Value after change

Effective date

Effective date of change

 

The file can be downloaded in csv format.
