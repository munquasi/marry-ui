Create New Payroll

You can create a payroll which can be configured to run on a regular basis (Monthly, fortnightly, weekly). when it runs on the scheduled time will calculate the salaries, taxes, deductions etc automatically based on the pay profile linked to it and other configured parameters.

 

To create a payroll:

  1. Click on the Payroll Administration and then select Pay Roll option.

    2.  

  2. Click the Create New Payroll button to create a new payroll.

 

 

  1. Fill in the desired payroll details as required.

 

 

While creating a payroll, you can configure the following:

  • Set the pay period for the payroll rule.

  • Schedule the created payroll to run automatically.

  • Select a particular pay profile for this payroll.

To set the Pay period for a Payroll:

 

  1. Provide the name of the payroll being created and set the Pay Period as monthly, fortnightly or weekly.

  2. Select the Shift Straddle rule which can be

  • Split shift

  • Shifts that start in the period

  • Shifts that end in the period

In case you select weekly or fortnightly, the Weekly Start Day and Weekly End Day will be displayed which you need to select.

. To configure Payroll run Rule

  1. In the Payroll Run section, enter the Days after End day information. For example if you enter 2 here, the payroll will be run 2 days after the end of the pay period.

  2. Enter the Time of day when the Payroll will be run.

  3. You have up to two run dates, a first run date and a second run date for the payroll.

  4. The Effective from Date allows you to set the date from which the payroll run is being considered.

To link pay profile to a Payroll

  1. You can select a pay profile from an existing list of pay profiles created in the payroll system .

     

Note: Payroll once run will come under the past payroll.
