One-Off Payroll

It allows payroll administrator to manually to generate pay for select pay profiles outside of the scheduled payroll run. This feature is useful for releasing salaries of individual or employees grouped under a profile in case their service is terminated or they leave the company before the scheduled monthly payment calculation.

 

For example, if the administrator manually runs a monthly profile (which is normally scheduled to run on 29th of month) on say 15th October, then this manual run will calculate pay till 15th of October (the date the one off payroll is run).

 

The scheduled payroll run will exclude this profile from the upcoming run for that month. (The next run scheduled on say 29th November will exclude this already run profile to avoid paying the employees again for this month.)

 

Single or multiple profiles payroll can be run in a one off pay profile.

 

The one off payroll allows Users to view the gross payroll file on the basis of on selected Employee(s), Pay Profile(s) or Payroll(s). For this, there are three tabs: Employee, Pay Profile and Payroll.

 

One off Payroll for Employees

 

  1. Click the Employee tab. The Employee section gets displayed as shown:

 

    

 

  1. Checkmark the desired Profile (s) displayed on the list, or search (if not displayed) and checkmark the Profile.

 

  1. Click the Run Payroll button to generate the XML file for the profile.

 

  1. The XML file will be generated and ready for viewing at your downloads folder.

 

    

 

One off Payroll for Pay Profile

 

  1. Click the Pay Profile tab. The pay profile screen gets displayed as shown:

 

    

 

  1. Checkmark the desired profile(s) displayed on the list, or search (if not displayed).

 

  1. Next, click the Run Payroll button to view the XML file.

The XML file will be generated for the pay profile and ready for viewing at your downloads folder.

 

One off Payroll for Payroll

 

  1. Click the Payroll tab. The Payroll section gets displayed as shown:

 

   

 

  1. Checkmark the desired Payroll (s) displayed on the list, or search (if not displayed).

 

As you checkmark the payrolls, the selected payrolls are listed below the results for reference.

You can either remove them by clicking the Remove link or proceed by clicking the Run Payroll button.
