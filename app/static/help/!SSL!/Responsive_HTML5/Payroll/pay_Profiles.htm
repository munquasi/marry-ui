Pay Profiles

The Pay Profile contains payment rules to be applied to a group of Employees. It specifies how the system will use the rules for calculating the pay of employees under that pay profile.

Create Pay Profile

You can create and configure a pay profile which you can attach to a payroll and run the payroll to generate pay for the employees coming under that pay profile.

 

To Create a Pay profile:

 

  1. Click on the Payroll module and then select Pay Profiles option.

          A screen with the list of existing pay profiles gets displayed.

 

          

 

 

  1. Click the Create New Profile button to create a new pay profile.

 

 

  1. Enter the pay profile details.

  2. Enter additional details that will be considered in the pay profile:

  3. Select the Effective From Date.

  4. Enter the Annual Leave accrual rate and annual leave guaranteed days.

  5. Select  the available pay rule that will be attached to this pay profile from the Core Basic option.

  6. Select  the available pay rule that will be attached to this pay profile from the Non Core Basic option.

  7. Click Save after entering all the relevant and mandatory information.

 

The description of the fields are as follows.

 

Field/Option

Description

Name

The name by which the Pay Profile is visually identified

Currency

The denomination that employees in this profile will be paid in

Period

The frequency that an employee will be paid. Weekly, fortnightly or monthly

Pay Indicator

Selector to determine where to derive the pay rate from:

●      Employee:  Pay rate as specified against the employee

●      Site: Pay rate as specified in the rate card associated with the post where the shift was completed

●      Higher: Compare the site rate with employee rate and chooses the higher of the two

Core Basic

The rule that determines pay for the basic hours work. The rules associated with this pay profile that will be applied when gross pay calculations are run against shifts for employees associated with this pay profile.

Core Basic - the rule that determines pay for basic hours work. Mandatory.

Non-core basic

The rule that determines pay for the basic hours work. The rules associated with this pay profile that will be applied when gross pay calculations are run against shifts for employees associated with this pay profile.
Non-core Basic- Other rules that can be applied for this this pay profile e.g. overtime and bank holiday rules.

Over time period hours

The threshold of cumulative work hours in a pay period that an employee must work before recording overtime hours. Threshold after which time is considered Overtime.

Probation Rate% Modifier

It is a modifier/calculation that is applied to the pay of an employee if that employee is on probation period.

Over time period

The duration within which the basic hours are accumulated. Weekly, fortnightly or monthly - can be different to Period.

Basic Hours

Normal working hours that does not include overtime,

Current Country

The country for which this pay profile is applicable.

Weekly Divider

The number of hours that will be used to calculate the total hours for a week.

Pay if Red Hours

In this case, the Employee may be paid despite the red hours being logged.

Red Hours: Under Working hours: If an employee has not completed the basic shift hours, then system will decide whether the employee will be paid for under working hours.

Always Paid Breaks

When check-marked, the employee linked to this paid profile will always get paid breaks even if the Allowance Card on that particular location is set to 'unpaid'.

Paid by Hours

Specifies whether the employee will be paid based on scheduled hours or actual hours. 

Over time period start day

Allows the setting of the day of the week the overtime will begin to be counted.

Effective from Date

Effective or start date for Pay Profile.

Annual leave accrual rate

Specifies the rate at which annual leave will be accrued to the annual leave balance of an employee.  Annual leave accrual rates is dependent on country. This rate is multiplied against number of days worked.

Annual leave guaranteed days

The number of annual leaves entitled to this profile.  The first 'x' amount of days will be automatically given (It can also be set to 0).

Reference Period

Reference period is the number of weeks we go back to calculate the average. So that in case an employee is not present, we still pay the employee based on the positive past record(the record is being calculated using this reference period).

Payrate

The Pay rate being used in case of leave. Whether Basic, Allowance or Overtime is being considered as the pay rate.

Worked Hour

The worked hour being considered in the leave conditions for this profile.

Absence Type

The Type of absence being considered in the leave being considered for this profile.

 

Attach Pay Rule to a Pay Profile

The user needs to map or attach a pay rule to a pay profile so that when that pay profile is attached in a payroll and run, the pay rules will be used in generating the pay file.

 

This explanation assumes that you have already created pay rules. For steps on how to create a Pay Rule, refer to section “Create Pay Rule”. For information on viewing a list of existing pay rules, refer to section, View Pay Rules View Pay Rules.

 

To attach a pay rule to a pay profile:

 

  1. Click on the Payroll module and then select Pay Profiles option.

          A screen with the list of existing pay profiles gets displayed.

 

          

 

  1. Click the Create New Profile button to create a new pay profile.

 

  1.  Enter the pay profile details.

 

  1. Scroll to the Available Pay Rule section and select a Core Basic pay rule.

 

          

 

  1. Next, select the Non Core Basic pay rule.

 

 

  1. Enter/select the other required fields and click Submit.

The selected Pay rule will be included in the newly created pay profile. The pay profile when included in a payroll will use the included pay rule while calculating the pay.

 

Edit Pay Profile

If the user has made no payroll runs using the pay profile, then the user can make changes to all the fields. However, if a payroll has been run even once using the pay profile, then the application restricts the amendments to only a few fields to ensure that no impact happens on existing payroll runs.

 

To edit or make amendments to a Pay profile:

 

  1. Click on the Payroll Administration module and then select Pay Profiles option.

      

  1. Click on any of the existing pay profiles to edit the pay profile.

 

 

The following screen displays:

 

 

  1. Make the required changes to the required fields and click the Save button to confirm the changes.
