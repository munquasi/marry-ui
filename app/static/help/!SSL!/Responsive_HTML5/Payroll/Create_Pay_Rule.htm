Create Pay Rule

User can create a pay rule with multiple conditions so that the system can generate the correct pay for a specific shift.

 

To create a payroll:

  1. Click on the Payroll Administration and then select Pay Rule option.

    2.  

  2. Click the Create New Rule button to create a new pay rule.

 

  1. Enter the name of the Rule.

 

 

  1. Choose the expenditure type (Basic Pay, Holiday pay, Overtime pay, Sick Pay).

  2. Select the rate amount from the Rate amount field.

  3. Select the pay element ID from the existing Pay Element ID list.

  4. Enable or disable the Core Basic pay option as required

  5. Enable or disable Include in Average Pay Calculation for absences as required.

  6. Select the the Effective From Date.

  7. Select the the Pay Rate.

  8. Enable or disable the Do not Pay if in Probation Period option.

  9. Select your desired Rate % Modifier from the list.

  10. Select the Hours Paid (which can be Contracted Hours, Scheduled Hours, Average hours).

  11. Select the Days of service from the Days of service field.

  12. Next, for the Shift selection section, fill in the shift details that will be applicable for this rule such as:

  • Time of day

  • Normal Days of the Week

  • Bank Holiday Days

  • Recurring

  • Set Specific Dates

  • Period

  • Number Of Consecutive Days Required

  • Range Of Days

  • Minimum Qualifying Hours

  • Maximum Qualifying Hours

 

     

Note: If we set period in pay rule as weekly, then the system will every time check from value 0 on start of the new period. So in the above example, system will calculate 12 minimum hours worked by the employee in a week starting from Monday onward & after the week is completed from next Monday onward it will start checking from 0 to 12 hours.

 

 

 

 

 

 

     

Note:

Minimum Qualifying Hours: If the user has entered any value in this field, then this rule will be only triggered for shifts meeting the minimum hours required data.

Min Hours Paid: If this rule is applicable to a shift, the employee will get paid at least for hours defined here even if he has worked less than the hours defined here.

Applicable Hours: In this case, this particular shift is applicable to the length of shift defined in the rule.

Set Specific Dates: This field will be used to create rule for a specific date.This rule will be triggered only if the shift is done the date mentioned in this field

 

 

 

 

 

 

 

 

 

 

  1. Next, fill in the Pay conditions such as:

  • Assignment Type

  • Time Type (overtime or basic)

  • Minimum Hours Paid

  • Applicable Hours

  1. After you have filled in all the relevant fields according to your requirement, click Submit on the upper right corner of the screen.

After the user has created the pay rules, these pay rules can be used in the pay profile while creating or updating a pay profile.
