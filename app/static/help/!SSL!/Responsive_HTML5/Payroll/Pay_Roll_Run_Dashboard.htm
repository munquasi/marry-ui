Pay Roll Run Dashboard

Users can view a dashboard of the past, current and future payroll runs and select a payroll run so that adjustment or alteration can be made to a payroll run.

 

After the payroll runs have been set and completed, the user can view the payroll run dashboard. The dashboard displays 12 months of completed runs (past) and 6 months of scheduled runs (future). so that the user can make adjustments or alterations to a payroll run.

 

The dashboard maintains a history of the payroll runs that allows you to view the payroll runs in the current period and past period.

View Current Payroll Run

The current payroll runs displayed on the dashboard is for the current pay period only. These payroll runs are those which are not yet submitted to the Payroll/Finance.

 

The filter/search allows the user to search the payroll runs on the basis of the any parameter displayed on the screen.

 

To view Current Payroll Run dashboard:

 

 

  1. Once you click on the Payroll Run Dashboard option, the dashboard is displayed (as shown below).

 

 

 

  1. Click Download against the desired payroll run record on the Payroll Run dashboard to download the pay file in XML format.

 

The current payroll runs displayed on the dashboard is for the current pay period only. These payroll runs are those which is not yet submitted to the Payroll/Finance.

 

The system displays 12 months of completed runs (past) and 6 months of scheduled runs (future) on the dashboard.

 

View Past Payroll run Dashboard

Once the Payroll is run and submitted to the finance (payroll), the history of all the previous runs are maintained in the Payroll Dashboard under past payroll section. The user can view the past payroll runs for the last 12 months. User can only view the previous payroll runs and they are not editable.

 

To view past Payroll Run dashboard:

 

    

  1. Click on the Pay Roll Run Dashboard option, the dashboard is displayed. By default it will open Current Tab.

 

    

 

  1. Click the Past tab on the Payroll Run dashboard.

 

   

 

  1. Click Download against the desired payroll run record on the Payroll Run dashboard to download the pay file in XML format.

 

   

 

The payroll runs can be filtered on the basis of the any parameter which are displayed on the screen.

 

The user can download the Payroll run record file in the form of XML format and it allows the user to access the payroll file submitted to Payroll.

 

View Future Payroll run Dashboard

The future payroll runs are the runs scheduled for the coming 6 months. The user can view the future payroll runs for the next six months.

To view the future Payroll Run record:

 

  1. Click on the Pay Roll Run Dashboard option, the dashboard is displayed. By default it will open Current Tab.

 

   

 

  1. Click the Future tab on the Payroll Run dashboard.

 

    

 

The payroll runs can be filtered on the basis of the any parameter which are displayed on the screen.

 

The payroll run record file is in the form of XML file and it allows the user to access the payroll file submitted to Payroll.
