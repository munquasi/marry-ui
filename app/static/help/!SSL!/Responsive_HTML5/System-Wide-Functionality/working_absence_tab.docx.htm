OPS-7-7.1.15 Working Absences

A working absence is the scheduled or unscheduled time off from work that occurs when an employee is not present at work during a normally scheduled work period.

 

This tab explains the following details about a tolerance.

●      Category of the Working Absence

●      Type of the Working Absence

●      Remarks for the working absence

●      Effective Date of the working absence

●      Expiry Date of the working absence

●      Active Status of the working absence

Ops-7-7.1.15.1 Add Working Absence

The working absence can be created anytime in the country module depending upon the requirement and urgency.

 

To add a new working absence:

 

1.     Click  icon form the Javelin Home screen and then select working absences from the drop down menu.

 

 

The following screen will appear on the screen.

 

 

2.     Click on the Add New Absence tab to display the following screen.

 

 

  The descriptions of the fields in Create Working absence Screen are given below:

 

Field

Description

Absence Category

Displays the broad category of reason the employee is not working. It can be of following categories.

Paid leave

Unpaid leave

Annual leave

Family leave

Sick leave

Absence Type

Displays the sub-category of the reason for the employee to not work. For every Absence Category there are different Absence Types. Hence, select Absence Category before selecting Absence Type.

Remarks

Displays the any comment or note for the selected working absence.

Effective Date

Displays the date from which the working absence is applicable.

Expiry Date

Displays the date till which the working absence is applicable.

It should display the current status of the tolerance.

When the absence is set as active then the option for effective to (end date) is grayed out.

 

3.     Click on Save to save the information and Cancel to cancel it.

The following screen will appear on the screen.

 

 

2.     Click on edit to edit the selected working absence.

 

 

Follow Ops-2-2.2.15.1 Add Working Absence to know the detailed explanation for all the fields.

 

3.     Click on Save to save all the details and Cancel to cancel the details.
