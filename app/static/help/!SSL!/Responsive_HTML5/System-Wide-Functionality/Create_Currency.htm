Create Currency

On clicking  button, the user can add new currency by providing inputs for the fields present on Country Currency screen as shown below:

 

  

 

The description of fields present on the Country >Create screen given below:

 

Field

Description

Currency Type*

The currency which belongs to vendor’s region and acceptable in the system will be defined as the type of currency on the Currency Type field.

 

Click on the  drop down menu, the list will get displayed on the screen as shown below:

 

 

Once the user select appropriate option from the drop down, the selected option will get displayed on the Currency Type field.( Default- official currency of the country, acceptable- all other currencies accepted by country)

Currency Code

The short forms defined for the currency’s name will be added to the Currency Code field to be acceptable in the system.

 

 

Click on the Search for Currency Code option, to find the appropriate currency code.

Click on the button, to save the rule applied for the currency for that particular country.

Click on the button to cancel the rule applied for the currency for that particular country.
