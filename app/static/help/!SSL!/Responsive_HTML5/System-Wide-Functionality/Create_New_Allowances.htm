Create New Allowances

Based on the allowance added in the country module, the employee is paid and the customer is charged. This allowance can be edited or deleted anytime.

 

To create a new Allowance:

 

1.     Click  icon on the Javelin Home screen and then select the Allowances option.

 

 

 

The following screen will appear displaying all the allowances under the Country Module.

 

 

2.     Click on the + Create New Allowance tab to display the following screen.

 

 

3.     Click on drop-down button next to Allowance Type tab to select the allowance type you want to select.

 

 

Select HCM Skill Based if you want to apply an allowance based on someone's specific skill set.

Following screen will appear if you select HCM skill Based.

 

 

Select Time Based if you want to apply an allowance based on number of days per week the employee is working.

Following screen will appear if you select Time Based.

 

 

Select Generic if you want to apply an allowance in general terms, without specifying a skill type or number of day in a week.

Following screen will appear if you select Generic.

 

 

The following description table explains about the various tabs on the Create Country Allowance screen.

 

Field

Description

Name

The name of the country allowance to be created.

Allowance Type

The type of the country allowance which needs to be created.

 

Following three options will appear in the drop down menu:

 

1.     HCM Skill Based

2.     Generic

3.     Time Based

 

Based on these allowance types various tabs will open (broadly discussed above this table).

4.     Click on Save to save the information and Cancel to cancel saving the information.
