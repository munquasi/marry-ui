Site Administration

The address format for the selected country will be defined by the Site Admin.  For defining the fields of the address format  for the selected country,  follow the below mentioned steps:

 

Select the Site Administration option from the Country Management drop down menu. The following Address Format screen will appear:

 

 

On Address Format screen, click on the Search for Country drop down menu. The list of countries will get displayed as shown below:

 

 

Select the country from the drop down list and search the Add Address Attribute to proceed with the search for that particular country. On clicking Add Address Attribute field the drop down menu will appear:

 

 

 

If user tries to skip the Add Address Attribute field the following message will get displayed as shown below:

 

 

The user can add the new field as required by clicking on the  button present on the Address Format screen.  Select the field from the Add Address Attribute field drop down menu as shown below:

 

 

Once the user select the field to be added , click on the  the button. The selected field will get displayed at the end of all the listed fields present on the Address Format screen as shown below:

 

 

To delete the field click on the  option, present under the Action heading. The field will get deleted from the Address Format screen as shown below:

 

 

As displayed on the above screen, the Longitude and City field is removed from the Address Format list.

 

The user can mark the fields as mandatory field by checking the  box present on the Address Format screen.

 

Once the user add all the fields for the Address Format and click on the  button the Address Attribute list will get saved.
