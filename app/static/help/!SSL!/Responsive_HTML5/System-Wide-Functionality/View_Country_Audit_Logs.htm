View Country Audit Logs

The user can check the Audit Logs at the country level to gather information for audit purposes. This tab will explain in brief about the current status of the selected Audit Log. It also shows the changes made so far in the selected Audit Log.

 

To view an Audit Log:

 

  1. Click  icon on the Javelin Home screen and then select the Country Audit Logs option.

 

 

The following screen will appear displaying all the audit logs in the country module.

 

 

 

2.     Select the audit log you want to view after looking under the Action Column.

      There are four kinds of audit logs which are defined under the Action column of the Audit Logs:

  • Save Duress Code

  • Update Duress Code

  • Delete Duress Code

  • Schedule employee

 

      Following screen will appear when you select audit log under the action Save Duress Code.

 

      

 

      Following screen will appear when you select audit log under the action Update Duress Code.

 

     

 

     Following screen will appear when you select audit log under the action Delete Duress Code.

 

     

 

    No screen will appear when you select audit log under the action Schedule Employee.

 

    The descriptions of the fields in Country Audit Logs Screen are given below:

 

Field

Description

Audit Log ID

Displays the ID assigned to the audit log.

Created by

Displays the Email ID of the person who created the audit log.

Created Date

Displays the date on which the audit log was created.

Changed by

Displays the Email ID of the person who last modified the audit log.

Last Modified Date

Displays the date on which the audit log was last modified.

Action

Displays the action taken for the audit log. There are three types actions under the audit logs under the country module.

 

1.     Save Duress Code

2.     Delete Duress Code

3.     Update Duress Code

4.     Schedule Employee

User Email

Displays the Email ID of the user related to the selected audit log.

IP address

Displays the IP address of the computer from which the audit log was created.

Reason for change

Displays the reason for which the audit log was last modified.

Namespace

Displays the ISO code of the country for which the audit log is created. Every country has its own unique ISO code.

Start date

Displays the start date of the selected audit log.

End date

Displays the end date of the selected audit log.

Service Line

Displays the area of work in which the employee is working.

Country Name

Displays the name of the country for which the audit log is created.

Duress Code

Displays the duress code related to the selected audit log.

Message

Displays a special message (if required) for future reference of the selected audit log.

Active Status

Explains whether the given audit log is active or not.

 

1 = Active

0 = Inactive

Delete Status

Explains whether the given audit log is deleted or not.

 

1 = deleted

0 = not deleted

 

3.     Click Close once all the details have been viewed.
