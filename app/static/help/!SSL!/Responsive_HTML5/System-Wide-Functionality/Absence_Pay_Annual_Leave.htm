Absence Pay: Annual Leave

This screen allows the configuration of various types of absence pays based on country. This ensures correct adjustments are made for the hours that are marked with 'absence type' as configured here.

 

To configure various leave adjustments:

  1. Click on icon and then select the Absence Pay menu.

 

           

  1. The following screen gets displayed on clicking Absence Pay menu.

 

 

  1. Select or enter the required information and click Save to apply the Absence pay.

 

Absence pay is dependent on the country and will firstly determine if an average No. of hours & Rate of pay is used.

 

Fields and options appearing in the screen

Field

Description

Reference Period

Reference period is the number of weeks we go back to calculate the average. So that in case an employee is not present, we still pay the employee based on the positive past record (the record is being calculated using this reference period).

Payrate

The Pay rate being used in case of leave. Whether Basic, Allowance or Overtime is being considered as the pay rate.

Worked Hour

The worked hour being considered in the leave conditions for this profile.

Absence Type

Type of absence applicable which is used to determine the number of payable hours and rates for each absence type.

Number of days country applicable

It is the number of days the country average pay rule is applicable
