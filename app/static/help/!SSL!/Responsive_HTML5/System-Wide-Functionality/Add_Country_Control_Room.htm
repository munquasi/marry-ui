Add Country Control Room

 

On clicking  button, the user can add new control room  by providing inputs for the fields present on Add Country Control Room screen as shown below:

 

 

 

 The description of fields present on the Add Country Control Room screen are given below:

 

Field

Description

Control Room Type*

The rules applicable to the control room for one default country will be set as default in the system as the Control Room type.

 

Click on the  drop down menu, the list will get displayed on the screen as shown below:

Once the user select appropriate option from the drop down, the selected option will get displayed on the Control Room Type field.

Control Room Name*

The set of rules which are predefined for Control Room and can be marked as default /others for one country.

 

On the Control Room Name field, mention the name of rules entitled for that particular Country Control Room.

Country Code

Country calling codes or country dial in codes are telephone dialing prefixes for the vendor countries or regions.

By clicking on the  icon, the list will get displayed on the screen as shown below:

 

 

Once the user select appropriate code from the drop down, the selected code will get displayed on the Country Code field.

Control Room Number*

The Control room number is a number used to contact the control room.

 

Mention the number  assigned to contact the Control Room of that particular country.

Click on the button, to save the rules and number applied for Control Room of that particular country.

Click on the button to cancel the process.
