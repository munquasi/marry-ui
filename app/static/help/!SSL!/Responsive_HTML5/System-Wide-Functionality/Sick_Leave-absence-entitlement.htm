Sick Leave

An employee's maximum sick leave entitlement for an employee needs to be automatically updated after a given period of days. Here, multiple rules set up in the Javelin application for each employee.

 

For example:

 

Maximum Sick Leave days paid = 28 After 0 days Service.

Maximum Sick Leave days paid = 30 After 365 days Service.

Maximum Sick Leave days paid = 35 After 1065 days Service

 

Edit Sick Leave

The user can edit the existing sick leave by clicking on the  option present at the end of each listed sick leave rule. On clicking the on the  option present at the end of each listed sick leave, the following Edit pop-up screen will appear:
