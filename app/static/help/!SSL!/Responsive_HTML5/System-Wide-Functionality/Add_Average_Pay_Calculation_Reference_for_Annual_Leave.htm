Add Average Pay Calculation Reference for Annual Leave

The administrator can set the Average Pay calculation Reference for annual leave for a selected employee.

 

The administrator can set the method for calculation of pay for selected type of leaves such as  compassionate leave, study leave, suspension, public duty and so on (using the Absence Type drop down).

 

To configure this:

 

  1. Click icon and then select the People menu.

 

 

A list of employee records gets displayed on a screen.

 

  1. Against an employee record, click . The information of the person gets displayed.

 

  1. The View Employee screen with Employee Info tab is displayed by default as shown below.

 

 

  1. Click the Leave tab on the View Employee screen. By default, the Sick Leaves tab for the selected employee gets listed:

 

 

  1. Click the Annual Leave tab. The following screen gets displayed

 

 

  1. In the screen, click the Calculation button.  The following screen displays:

 

 

  1. Select the absence type, threshold hours, reference period, and the sick leave parameters.

 

  1. Click Save after providing the required details.

 

Field

Description

Absence Type

Type of absence

Minimum Threshold Hours

The hours to considered when the average hours calculated is below the minimum threshold.

Maximum Threshold Hours

The hours to considered when the average hours calculated is below the maximum threshold.

Reference Period

Reference period is the number of weeks we go back to calculate the average. So that in case an employee is not present, we still pay the employee based on the positive past record(the record is being calculated using this reference period).

Rate % Modifier

If set, the employee rate will be multiplied by this % to derive the rate to apply

Range of Days

Min: Allows the user to give a minimum number of days worked if average calculation is less than min threshold days.

Max: Allows the user to set the maximum number of days worked if average calculation is more than maximum threshold days.

 

  1. Add or update the details in the desired fields.

 

  1. Click Save to proceed with the creation.
