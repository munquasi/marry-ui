View People's Calendar

The People’s calendars are a very efficient organizer tool for any department. It helps in sharing up-to- date information about upcoming events and scheduled holidays while significantly reducing company-wide emails. It helps the managers to track and manage the events attached to any employee reporting to that manager.

 

To view people’s Calendar:

 

  1. Click icon and then select the People menu

 

.

 

A list of employee records gets displayed on a screen.

 

  1. Against an employee record, click . The Calendar of the person gets displayed.

 

 

  1. The calendar view allows the user to view the existing schedules of the user including various events lined up in the selected months view.

 

          The user can view the calendar of a person on the basis of Today, Day, Week, or Month.
