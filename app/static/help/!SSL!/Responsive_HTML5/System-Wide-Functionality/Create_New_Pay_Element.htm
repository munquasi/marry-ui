Create New Pay Element

New pay elements can be only be created in the country module with the help of Create New Pay Element tab.

 

To create a new Pay Element:

 

  1. Click  icon form the Javelin Home screen and then select Pay Element.

 

 

The following screen will appear displaying all the pay elements under the Country Module.

 

 

 

  1. Click on Create New Pay Element to create a new pay element details.

The following screen for creating a new pay element appears on the screen.

 

 

      The descriptions of the fields in Create New  Pay Element Screen are given below:

 

Field

Description

Expenditure code

It is the alphanumeric code which is fetched from finance during costing procedure and is reference data from Oracle ERP.

Pay Element Code / Pay Element ID

Pay Element code is an alphanumeric code given to the pay element being defined.

 

Pay element code is reflected as pay element ID in the list of pay elements when you select pay element from the country module.

Description

Description of the pay element being defined.

Effective from

Date from which the pay element will be considered.

Effective to

Date till which the pay element will be considered.

 

  1. Click on SAVE once everything is filled correctly or CANCEL to cancel the Create New Pay Element tab.

The following pop-up will appear on the screen.

 

 

  1. Click OK to go back to the list of  all the pay elements on the country module.
