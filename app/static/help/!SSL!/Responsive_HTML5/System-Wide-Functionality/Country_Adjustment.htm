Country Adjustment

This module allows the system administrator to set up the default configuration required to make adjustments to employee payments across in the system.

Edit Existing Adjustment Head

You can amend or change an adjustment of the employee salary if required.

To edit an existing Adjustment Head:

 

  1. Click on Country Management and then select the Adjustment menu.

 

            

 

  1. The Adjustment screen gets displayed on clicking Adjustment menu.

 

  1. Click the option against the Adjustment that you want to edit.

 

 

  1. Make the desired changes to the information as required.

  2. Click Save after making the required changes.

 

 

Delete Existing Adjustment

You can delete an existing adjustment head if required.

 

To delete an existing Adjustment:

 

  1. Click on Country Management and then select the Adjustment menu.

 

          

 

  1. The Adjustment screen gets displayed on clicking Adjustment menu.

 

  1. Click the    option against the Adjustment that you want to delete. The following message displays

 

 

  1. Click Confirm to proceed with the deletion. A success message gets displayed.
