Add New Holiday

A holiday can be added anytime in the country module.

 

To add a new Holiday:

 

  1. Click  icon on the Javelin Home screen and then select the Holiday Calendar option.

 

 

The following screen will appear displaying the Holiday List tab of the country module.

 

 

  1. Click on Holiday to display the following screen.

 

 

  1. Click on Add New Holiday tab.

The tab for adding a new holiday will appear on the screen.

 

 

By default the start time and end time will be 12:00 AM. You can change it too (if required).

 

The descriptions of the fields in Create New Holiday Screen are given below:

 

Field

Description

Holiday Name

This will be reflected from the holiday list created in the country module.

The selected holiday from the drop down list will appear in the holiday name tab.

Holiday date

 

The date of the holiday being mentioned.

You can select the date from a drop down menu or type the date in DD/MM/YYYY format.

Fixed / Not Fixed

System Admin can define the holiday to be fixed for the selected date every year.

 = not fixed

   = fixed

 

  1. Click on SAVE to save the information or CANCEL to cancel the Add New Holiday tab.
