Holidays List

This tab provides the list of holidays at the country level. The user can add, edit and delete holidays in the holiday list at any time. The code of holiday list can also be changed at any time to avoid confusions for new added holidays in the holiday list.

 

The holiday list can be added anytime by the system admin.

 

To add a new Holiday List:

 

  1. Click  icon on the Javelin Home screen and then select the Holiday Calendar option.

 

 

A screen will appear displaying the Holiday List tab of the country module.

 

  1. Click on Add New Holiday List tab. The tab for adding a new holiday list will appear on the screen.

 

 

  1. Enter the name of the holiday in the Holiday Name tab. This name will be displayed in the list of holidays while selecting Holidays tab in the Holiday Calendar tab.

  2. Type a code for the holiday in the Holiday Code tab. This code can be alphanumeric too.

  3. Click on SAVE to save the information or CANCEL to cancel the Add New Holiday List tab.
