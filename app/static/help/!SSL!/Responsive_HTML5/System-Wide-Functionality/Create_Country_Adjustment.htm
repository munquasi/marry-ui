Create Country Adjustment

Here, you can create the adjustment Heads and map it with their associated adjustment parameters to use them in other related screens for making the various adjustments to the employee payments and salaries.

 

To Create a country Adjustment:

 

  1. Click on Country Management and then select the Adjustment menu.

 

      

  1. The Country Adjustment screen gets displayed on clicking Adjustment menu.

 

  1. Click on  to create a new adjustment. The following screen displays:

 

 

  1. Enter the Adjustment Head Description.

  2. Select one of the options under Is Deductible (it can be addition or subtraction) drop down list.

  3. Select the Pay element ID from the Pay Element ID dropdown list.

  4. Select the expenditure Code from the Expenditure code list.

  5. Click Save after entering the required details to confirm the information just created.

 

 

For an existing Adjustment:

●      Click the  option to edit an adjustment.

●      Click the option to delete the adjustment from the existing list.

 

 

Fields

Descriptions

Adjustment Head Description

Name of the adjustment by which it will be known

Is Deductible

Whether this is a deductible adjustment

Expenditure Code

It is fetched from finance during costing procedure and is a reference data from Oracle ERP.

Pay Element ID

This information helps to identify the type of payment incurred (whether it is for holiday, overtime, etc). The Pay element ID is created in the Country > Pay Element section of the application. If you want to know the pay element ID, you can just navigate to this screen and look up the existing values.

 

The country adjustment heads created here will be in other screens of the application to make adjustments to employee payments and salaries, and other expenditures if required.
