About Operations

The Javelin application provides a seamless integration between HR, Operations and Finance to facilitate management of all the employee related information of G4S.

 

The Javelin application stores the employee’s detailed information and makes it available across the various departments of the organization. It includes advanced technology, smooth integration, and caters all the activities involved in operations, starting from adding Customer, initiating Contract, getting Work Order, locating Sites, Post, payrolls,  and managing Shifts to accurately forecasting future workforce needs.

 

The Javelin OPS:

 

  • Enables managers to record, track, monitor and manage information and actions relating to Customers, and Security Officers.

  • Facilitates smooth management of customers by facilitating the management of contracts, billing, work orders, posts, catalogues and so on.

  • Allows management of security officers, via contracts management, payrolls,  scheduling, time and attendance, skills, licenses, posts, training, and so on.

  • Covers various level of users and protects the confidentiality, essential to every organization.

  • Provides a rich array of analytical reports related to customers and security officers that allows the authorized managers to analyse data and trends, and then take key decisions and actions.

Purpose

This User Manual introduces the basic layout and functions and features  of the Javelin application suite. It enables users to proactively manage operations while focusing on strategic business initiatives.

Target Audience

This work instruction will guide the G4S Employee to work on the Javelin application suite independently.

Accessing Javelin application

To start with the Javelin application suite, the user needs to request for application credentials and access as per the role assigned by sending an email to the concerned department, with the supervisory approvals.

 

Once approval is received, the user can log into the application with the G4S email ID and password provided by the Admin.

 

For accessing Javelin application suite, the user needs to send a request to the Admin through email with the required details mentioned (like user’s department or division, phone number and supervisory (Manager’s or Higher authorities) approvals or permissions).

 

Browser Requirement

The Javelin application is a web based application and can be accessed through Chrome or by Internet Explorer 6.0 and higher.
