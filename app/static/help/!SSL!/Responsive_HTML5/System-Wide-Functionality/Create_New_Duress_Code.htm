Create New Duress Code

On clicking  button, the user can add new duress code by providing inputs for the fields present on Country Duress Code screen as shown below:

 

 

 The description of fields present on the Country Duress Code screen are given below:

 

Field

Description

Service Line*

Select the from the drop down menu.

 

Select the appropriate service line, the result will get displayed on the Service Line field.

Duress Code*

Duress Code is the unique code and will be used in case of emergency. The duress code is specific to service line.

Start Date *

The date from which the duress code will remain active. Click on the  icon, the calendar to select date will  get displayed:

 

 

Once the user select the Start date, it will get displayed on the Start Date field.

 

End Date

It is the date on which the duress code will become inactive and stop sending alerts. Click on the  icon, the calendar to select date will get displayed:

 

Once the user selects the end date, it will get displayed on the End Date field.

 

Active

The Active checkbox is displayed as checked by default. If user mark it  'unchecked', then the 'Effective End Date' field will get enabled and is mandatory to mention the expiry date.

Click on the button, to save the Duress Code

Click on the button to cancel the process.
