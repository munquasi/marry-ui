Create New Wage

The Minimum wages tab of the Country modules defines the wages given to the employees of G4S.

New wages can be created for the employee with the help of Create New Wages tab. The wages given to the employees varies from age to age.

 

To create a new Wage:

 

  1. Click  icon on the Javelin Home screen and then select the Minimum Wages option.

 

 

The following screen will appear displaying all the minimum wages under the Country Module.

 

 

  1. Click on Create New Wages to create new wage details. The following screen for creating a new wage appears on the screen.   

 

 

       The descriptions of the fields in Create New Wage Screen are given below:

 

Field

Description

Start Age

The acceptable start age for the wage being defined. It should be minimum 16 years.

End Age

The acceptable upper age for the wage being defined.

It should be maximum 80 years.

Wages

Wage that will be applicable and disbursed for this wage group.

Effective from

Date from which the wage calculation will be considered.

Effective to

Date till which the wage calculation will be considered.

 

  1. Click on SAVE once everything is filled correctly or CANCEL to cancel the Create New Wage tab.

 

The Start Age and the End Age should be between 16 to 80. If you enter an age not in this range then a warning message will appear on the screen as shown below.

 

The system will not allow a new record with already existing date range or age range.
