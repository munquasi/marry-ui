Annual Leave

An employee's maximum annual leave entitlement for an employee needs to be automatically updated after a given Period of days. Here, multiple rules set up in the Javelin application for each employee.

 

For example:

 

  • Maximum Annual Leave days paid = 28 After 0 days Service

  • Maximum Annual Leave days paid = 30 After 365 days Service

  • Maximum Annual Leave days paid = 35 After 1065 days Service

 

The Country Management module will allow employee, to define absence entitlement for annual leaves on the basis of  'Number of days service required'.

 

 

Edit Annual Leave

The user can edit /delete the sick leave by clicking on the  icon present at the end of each listed annual leave rule. On clicking the on the  option present at the end of each listed annual leave, the following Edit pop-up screen will appear:
