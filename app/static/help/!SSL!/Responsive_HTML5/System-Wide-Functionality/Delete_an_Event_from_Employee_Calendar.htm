Delete an Event from Employee Calendar

You can delete a scheduled event from an employee’s calendar if at any time, there is a requirement to do so.

 

To delete an event from an employee’s Calendar:

 

  1. Click icon and then select the People menu.

 

 

A list of employee records gets displayed on a screen.

 

  1. Against an employee record, click . The Calendar of the person gets displayed.

 

 

  1. Double-click on the any event (marked with color) to delete an existing event from the calendar of the employee. The following screen displays.

 

 

The details of the existing event gets displayed as shown.

 

  1. Click Delete to delete the event from the list of events in the employee’s calendar. The following screen displays:

 

 

  1. Click Confirm Delete to proceed with the delete from the employee’s calendar.
