Activate a Role

 

To activate a Role:

 

  1. Click icon and then select the Create Roles option.

 

 

  1. Next, click on any of the existing roles on the resulting screen.

 

  1. Against the record you want to edit, click the  link. The record expands in the same screen in edit mode as shown:

 

 

  1. Enable the Active option and click Save. The following screen displays:

 

 

  1. Click Confirm to proceed with the activation and save the change.

 

The user may require a certain privilege to activate a role.

 

In case the message “Full authentication is required to access this resource” is displayed while trying to activate a role, please contact your administrator.
