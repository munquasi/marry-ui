Configuring/Updating Menus for Role

The System Administrator can view the list of available menus in the user interface and configure for each role, the menu items to be shown in the left hand bar in the Javelin OPS application.

 

Then by selecting the check boxes next to each of the menus you will grant or remove access to these menus for this role.

 

To Configure/update Menus for a Role:

  1. Click icon and then select the Roles Administration:Menus menu.

 

 

  1. The following screen is displayed on clicking the Roles Administration: Menus option.

 

 

  1. From the Roles drop down list: select the role whose menu you would like to configure.

 

 

  1. For the selected Role, the list of menu items are displayed against which the logged in user can enable or disable the access.

 

Menus and menu items are configured in a parent / child relationship. When the parent is checked, the child items will automatically be checked, unless otherwise overridden.

 

 

  1. Then by selecting the checkboxes next to each of the menu items listed, you can grant or remove access to these menus for this role.

  2. Click the Save button to confirm the changes.

 

Once a change has been made you must click Save in the top left hand side of the screen to save the changes.

 

When you select a Role and then begin to make changes to the menu permissions, an Undo Changes button will appear. This is so that BEFORE you press save, it is possible to restore the permissions to the previously saved version.

Note: once you press save, the Undo Changes is no longer possible.
