Update a Job to Role Mapping

The System Administrator can update the mapping of a new role from the Least Privileged User to another role.

 

If a Job Title is updated in HCM in the reference list of Job Titles, then that Job ID will be remapped to the Least Privileged User role automatically. The System Administrator would need to then log into the system and update the mapping to the relevant role.

 

To update job title to role mapping:

  1. Click icon and then select the Job Title to Role Mapping menu.

 

 

  1. The following screen is displayed on clicking the Job Title to Role Mapping option.

 

 

  1. Click the  icon against a record that belongs to the least privileged user. A screen displays with various fields that allows the logged in user to update the required role.

 

 

  1. Select the required role to be mapped. The user can also edit the comments (if required). Click Update. The following confirmation message displays.

 

 

The System Administrator can update the mapping from Least Privileged User to another role, (which was created on the Create Roles page).

 

  1. Click Confirm. The role selected will be updated for the job profile.
