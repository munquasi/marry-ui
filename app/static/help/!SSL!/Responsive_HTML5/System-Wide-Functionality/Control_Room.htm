Control Room

This master defines the control room number for a country. To define the rule for Country Control Room follow the steps mentioned below:

 

Select the Control Room option from the Country Management drop down menu. The following Control Room screen will appear:

 

 

 

Edit Country Control Room

The user can edit country control room details by clicking on the  option present at the end of each listed Control Room Type. On clicking  option, the user will be allowed to edit the selected row. The fields of the selected row will get edited as shown below on the Edit screen:

 

 

Once the user edit the information and click on the  button the changes will get  saved and on clicking   button the changes will not get updated.

 

Delete Country Control Room

The user can delete the control room by clicking on the  option present at the end of each listed control room type. On clicking  option, the pop-up message to confirm the control room deletion will appear:

 

 

On clicking the  button, the rule applied for control room will get deleted from the Control Room Type rule list. To cancel the deletion process, click on the  button.
