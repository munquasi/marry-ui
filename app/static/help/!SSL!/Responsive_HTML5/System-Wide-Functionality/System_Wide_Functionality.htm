System Wide Functionality

The System Wide Functionality pillar of Javelin application suite is designed to cater all the major requirements of Javelin application.  Only System Administrator has the rights to login System Wide Functionality pillar of the Javelin application.

 

The System Wide Functionality includes the following modules and sub-modules:

 

1.    Country Management

  1. Minimum Wages

  2. Break Management

  3. Holiday Calendar

  4. Allowances

  5. Tolerance

  6. Absence Entitled

  7. Currency

  8. Control Room

  9. Site Administration

  10. Duress Code

  11. Organization Hierarchy

  12. Count Audit Logs

  13. Adjustment

  14. Absence Pay

2.     System Administration

3.     Directory

The user can define the master in the Country Management module, for one country wherein application is deployed. These masters will be consumed by other pillars to fetch the information. The Country Management module is visible to System Administrator only.
