Bank Holiday (Average Pay Calculation Reference)

The administrator can set the Average Pay calculation Reference for Bank Holiday for a selected employee using this screen.

 

To Set the Average Pay calculation Reference for Bank Holiday:

 

  1. Click icon and then select the People menu.

 

 

A list of employee records gets displayed on a screen.

 

  1. Against an employee record, click .

  2. The View Employee screen with Employee Info tab is displayed by default as shown below.

 

 

  1. Click the Bank Holidays tab on the View Employee screen. The Bank Holidays parameters setting screen gets displayed along with relevant details:

 

 

  1. Set the values of the parameters and click Save to confirm the information for the employee.

 

Field

Description

Minimum Threshold Hours

The hours to considered when the average hours calculated is below the minimum threshold.

Maximum Threshold Hours

The hours to considered when the average hours calculated is below the maximum threshold.

Reference Period

Reference period is the number of weeks we go back to calculate the average. So that in case an employee is not present, we still pay the employee based on the positive past record(the record is being calculated using this reference period).
