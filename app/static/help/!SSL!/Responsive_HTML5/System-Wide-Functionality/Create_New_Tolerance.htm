Create New Tolerance

The tolerance can be created anytime in the country module.

 

To create a new Tolerance:

 

1.        Click  icon on the Javelin Home screen and then select the Tolerance option.

 

 

The following screen will appear on the screen

 

 

2.        Click on the Create New Tolerance tab to display the following screen.

 

 

 

 

         The descriptions of the fields in Create Country Tolerance Screen are given below:

 

Field

Description

Tolerance Name

Type the name of the tolerance.

Service Line

Select the service line from the options given in the drop down menu.

 

 

The selected service line will be displaced in the service line box.

Before shift start

Type the allowed tolerance before the shift starts.

After shift start

Type the allowed tolerance after the shift starts.

Before shift Ends

Type the allowed tolerance before the shift ends.

After shift Ends

Type the allowed  tolerance after the shift ends.

Before Call

Type the allowed  tolerance before the call starts.

After Call

Type the allowed  tolerance after the call starts.

Start date

 

Select the start date for the given country tolerance.

 

 

End date

Select the end date for the given country tolerance.

 

Active / Inactive

It should display the current status of the tolerance.

 

       = inactive

      = active

 

 

When the tolerance is set as active then the option for effective to (end date) is grayed out.

 

3.     Click on Save to save the information and Cancel to cancel it.
