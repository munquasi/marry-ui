Country Allowances

System Admin can define different types of allowance for a country. these can be on skill type, time based or generic or hcm based. Along with the Allowance Name, this tab explains the Allowance type and Skill Type under the same tab. This way it becomes easier to fetch information for each allowance.

 

Edit Existing Allowances

Any allowance can be edited at any time under the country module.

To edit an existing Allowance the following steps are required.

 

To edit a new Allowance:

 

  1. Click  icon on the Javelin Home screen and then select the Allowances option.

 

 

The following screen will appear displaying all the allowances under the Country Module.

 

 

2.     Click on Edit  to edit the selected Allowance.

 

In this example we will consider the allowance with allowance type as Generic. But the process is same for Time Based and  HCM skill Based.

 

 

3.     Click on Save to save all the details and Cancel to cancel editing.

 

Delete Existing Allowance

There may be a requirement to delete an existing allowance at any time under the country module.

 

To delete an Allowance:

 

  1. Click  icon on the Javelin Home screen and then select the Allowances option.

 

 

 

The following screen will appear displaying all the allowances under the Country Module.
