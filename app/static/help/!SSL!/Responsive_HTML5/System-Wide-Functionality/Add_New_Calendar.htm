Add New Calendar

A calendar can be added anytime in the country module.

 

To add a new Calendar:

 

  1. Click  icon on the Javelin Home screen and then select the Holiday Calendar option

 

 

The following screen will appear displaying the Holiday List tab of the country module.

 

 

  1. Click on Calendar to display the following screen.

 

 

  1. Click on Add New Calendar tab.

The tab for adding a new calendar will appear on the screen.

 

 

The descriptions of the fields in Add New Calendar Screen are given below:

 

Field

Description

Calendar Name

Name of the calendar being added.

Calendar type

 

The current status of the calendar. 

 

Select the calendar type from the drop down menu.

1.     Default

2.     Other

Holiday List

The list of all the holidays defined under the country module.

 

Along with the name of the holiday in the holiday list, It consists of  Holiday date, Start time and End time for which the holiday is defined.

Status

The current status of the Calendar.

         = inactive

       = active

 

 

  1. Click on Holiday List to add holidays in the holiday calendar.

 

 

  1. Select the holidays you want to add in the holiday calendar from the list of holidays by putting a check mark next to the holiday you want to add.

  2. Click on CONFIRM once the holidays are marked.

  3. Click on SAVE to save the information or CANCEL to cancel the Add New Calendar tab.

 

There could be only one calendar as default under the calendar type. Rest every calendar should be under the name others.

 

If the user tries to make any other calendar as a default calendar then the system will show a pop-up warning on the screen as shown below.

 

 

There cannot be any holiday calendar without any holiday in it.
