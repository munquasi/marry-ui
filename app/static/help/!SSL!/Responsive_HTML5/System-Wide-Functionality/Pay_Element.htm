Pay Element

The Pay Element tab of the country module helps to identify the type of payment incurred (whether it is for holiday, overtime, etc). The data entered in the pay element list is reflected in the following places:

●      Pay rules tab of the Payroll module

●      Adjustments tab of the Country module

 

 It displays the following information about a pay element.

●      Expenditure code of the pay element

●      Pay Element ID of the pay element

●      Description about the pay element

●      Effective dates of the pay element
