Duress Code

Once the system validates the control room number, a duress alert in the form of a Duress Code is decided and shared with the employee posted on that particular location.

 

The Duress Code is a unique code which is used to send an alert to the control room for an emergency situation arises at employeeâ€™s location. The employee can pass the duress code along with the control room number at the time of emergency. The Duress Alert pop up will appear on the dashboard of the Javelin application.
