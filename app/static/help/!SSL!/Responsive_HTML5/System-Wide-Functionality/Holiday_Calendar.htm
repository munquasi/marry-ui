Holiday Calendar

The Holiday Calendar tab provides information about the holidays mentioned in the country module. System admin can define country rules for holidays, create lists and can define different holiday calendars which can be applicable in the country.

 

Edit Existing Calendar

A calendar can be edited anytime in the country module.

 

To edit a Calendar:

 

  1. Click  icon on the Javelin Home screen and then select the Holiday Calendar option.

 

 

The following screen will appear displaying the Holiday List tab of the country module.

 

 

  1. Click on Calendar to display the following screen.

 

 

  1. Click on Edit to edit the required fields in the selected Holiday Calendar.

 

 

 

  1. Click on SAVE to save the information or CANCEL to cancel the Edit  Holiday Calendar tab.

 

The following message will appear on the screen once a calendar is edited.

 

 

Delete holiday calendar

A calendar can be deleted anytime in the country module.

 

To delete a Calendar:

  1. Click on Delete the delete the selected holiday calendar.
