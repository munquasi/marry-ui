Country Tolerance

The Country Tolerance is a master to define tolerance limits for employees across different service lines for a country. This tab explains the following details about a tolerance.

 

1.     Tolerance name

2.     Service line

3.     Tolerance before and after shift starts

4.     Tolerance before and after shift ends

5.     Tolerance before and after Check call

6.     Start date and end date of the tolerance

7.     Active status of the tolerance

 

Edit Existing Tolerances

Any tolerance can be edited at any time under the country module.
