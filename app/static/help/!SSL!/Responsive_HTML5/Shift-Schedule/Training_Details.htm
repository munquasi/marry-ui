Training Details

Training has specific goals of improving one's capability, capacity, productivity and performance. G4S provides the trainings to its employees at the place of work, while they are on the actual job. The training is scheduled/ planned keeping the business needs in mind whereby an individual acquires job-related skills and knowledge.

 

To view the Training Details for the employees, click on the Shifts tab present on the Navigation bar and select the Training Details option from the drop down menu. On clicking Training Details option from the Shift drop down menu present on the Welcome screen of Javelin application, the following Trainings screen will appear:

 

 

For viewing training details for a employee, mention the name of the employee/training  in search list  and click on the search option to find the record. The result  will get displayed on the bottom of the screen as shown below:

 

 

The description of fields present on the Trainings screen as follows:

 

Field

Description

P#

Displays the start time and end time of the shift.

Post Name

Displays the name of the post for which shift is scheduled.

Site ID

Displays the unique site id in numbers assigned to each shift

Site Name

Displays the name of the site assigned to each shift

Customer Name

Displays the name of the customer for which shift is scheduled

 

The user can view the Training details scheduled for the particular employee by clicking on the button present at the end of each employee name. On clicking button the following Training Details screen will appear:

 

 

 

On clicking button, the Training details screen will get closed and the user will get navigated to the Trainings screen.
