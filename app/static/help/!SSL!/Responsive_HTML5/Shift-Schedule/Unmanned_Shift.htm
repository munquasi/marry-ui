Unmanned Shift

When the shift is not manned due to unavoidable circumstances like: SO is off duty, Regular scheduling gap and Unplanned Scheduling gap then the shift is marked as Unplanned Shift.

 

To view the Unmanned Shifts, click on the Shift tab present on the Navigation bar and select the Unmanned Shifts option from the drop down menu. On clicking Unmanned Shifts option from the Shift drop down menu present on the Welcome screen of Javelin application, the following Unmanned Shifts screen will appear:

 

 

On the Unmanned Shift screen the shift details are displayed date wise. To check the shift scheduled for that particular date, click on the any link, listed on the screen. The information related to Shift, Post and Site will get displayed as shown below:

 

 

The descriptions of the fields present on Unmanned Shift screen are given below:

 

Field

Description

Shift Start/End Time

Displays the start time and end time of the shift.

Post Name

Displays the name of the post for which shift is scheduled.

Site ID

Displays the unique site id in numbers assigned to each shift

Site Name

Displays the name of the site assigned to each shift

Customer Name

Displays the name of the customer for which shift is scheduled

 

To manned the unmanned shift click on the icon present at the end of each listed shift. On clicking the icon, following Manual Scheduling screen will appear as shown below:
