1.     On clicking Recommendation tab present on the Shift scheduling screen.

the list of employees best suited to the empty shift will get displayed as shown below:

 

 

 

 

2.             Select the recommended employee, by clicking on the Accept icon present on the Recommendation tab.

On clicking Accept icon, the following Accepted screen pop-up will appear:

 

 

 

    The descriptions of the fields present on the Accepted pop-up screen as follows:

 

Field

Description

Start Time

Start time allocated to the Employee shift. Displays the time set as system’s current time.

 

Displays the current date by default on the Start Time field.

To change/select another start time for the shift, click on the icon.

End Time

Completion time allocated to the Employee’s shift. Displays the End time as per the time set for shift cycle, depending on the start time. The end time will be calculated on the basis of predefined hours for that particular shift.

Comment

Comment regarding the shift allocation and other related details.

Call out Allowance

Check the box if the allowance is applicable.

Allowance Type

Select the type of allowance applicable from the drop down menu.

Amount

The allowance amount entitled for the employee. It will reflect in Euro.

Calculation

Mention the amount applicable for this shift.

Click on the Save button to save the information.

Click on the Cancel button to cancel the process.
