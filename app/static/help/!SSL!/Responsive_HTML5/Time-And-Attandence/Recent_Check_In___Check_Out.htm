Recent Check In / Check Out

On clicking Recent Check In /Check Out option, the following Check In/Out Dashboard screen will appear:

 

 

       On Check In/Out Dashboard screen, the user can view the information on recent check in /check out along with employee information.  To view the employee information in detail, click on the Employee Name link listed on the Check In/Out Dashboard screen. The following Employee Information screen will appear:

 

 

When any employee do Check IN/Check Out, then the employee record will get displayed in Recent Check In/Check Out Dashboard screen instead of Control Room Dashboard screen.

 

        On the View Employee screen, the user can view the detailed information about the employee. The information like Personal Details, Assignment, Photo, Profile, Role, Visa Permits, Leave and Bank Holidays can be viewed by clicking on the tabs present on the View Employee screen.
